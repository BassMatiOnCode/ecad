PSpice for TI

PSpice for TI is the successor of TI TINA, for various reasons. The TI version is not limited in design size, but somewhat limited to TI models. Only three imported models can be probed simultaneously. This does not apply to power MOSFETs, power diodes, passives with parasitics, independent sources, switches and transformers.

  • Unlimited number of nodes
  • Multipage and hierarchical schematics

Includes standard PSpice libraries:

ABM

Analog behavioral models for various math functions

ANALOG

Passives, dependent sources, switches, transmission lines

BREAKOUT

Customizable versions of many device types

DIG_MISC

Digital timing control, pull-up / pull-down resistors

DIG_PRIM

Logic gates, flip-flops

SOURCE

Independent voltage and current sources

SPECIAL

Parameters, simulation control, library management, utilities

TI TINA

TI-TINA has been superseeded by PSpice for TI, for a number of reasons:

  • No parametric variable sweeps
  • No Monte-Carlo analysis
  • No worst-case analysis
  • 1300 vs 5700 internal libraries
  • No automatic library updates
  • No built-in modeling application
  • No hierarchical schematics
  • No multipage schematics
